Testando Media Queries

O QUE SÃO MEDIA QUERIES?

Media queries são recursos do CSS usados para adaptar o visual de um site de acordo com o tamanho e as características da tela em que ele está sendo exibido. Isso significa que, com media queries, é possível mudar o estilo de um site para que ele fique mais bonito, organizado e fácil de navegar em diferentes dispositivos, como celulares, tablets, computadores e até TVs.
Elas funcionam como uma “condição” que o navegador verifica: se a condição for verdadeira, o CSS que está dentro daquela media query será aplicado. Por exemplo, se uma media query estiver configurada para telas com no máximo 600 pixels de largura, as regras dentro dela só serão aplicadas em dispositivos pequenos, como celulares.

BREAKPOINTS MAIS USADOS

Os breakpoints são os tamanhos de tela que os desenvolvedores usam como referência para decidir quando mudar o layout do site com media queries. Eles são chamados assim porque representam o “ponto de corte” entre um tipo de dispositivo e outro. Os breakpoints mais comuns são:

  • Até 600px: Normalmente usados para celulares. É onde se faz a maioria das adaptações para telas pequenas, como reduzir fontes, empilhar blocos ou transformar o menu em ícone.
  • Entre 601px e 1024px: Faixa usada para tablets. O layout precisa ser mais espaçado que no celular, mas ainda não tão largo quanto no computador.
  • Acima de 1024px: Telas de computadores (desktops e notebooks). Aqui o site geralmente mostra o layout completo, com todos os elementos visíveis.
  • Acima de 1440px: Telas maiores, como monitores grandes ou TVs. Pode ser usado para aumentar margens, fontes e espaçamentos, se necessário.

COMO USAR MEDIA QUERIES NA PRÁTICA?

Na prática, media queries são muito úteis para tornar o site “responsivo”, ou seja, capaz de se adaptar automaticamente ao tamanho da tela do usuário.
Com elas, é possível reorganizar elementos do layout, mudar o tamanho de textos, esconder ou mostrar partes do conteúdo e até trocar imagens para versões menores (mais leves) em celulares. Por exemplo, um menu horizontal que aparece no topo do site no computador pode ser transformado em um menu hambúrguer (aquele ícone com três linhas) no celular, tudo isso controlado por media queries.
Outro exemplo prático é deixar imagens menores no celular para carregar mais rápido e economizar dados. Para isso, o desenvolvedor escreve no CSS uma media query que diz: “se a tela tiver até 600px, aplique este conjunto de estilos”. E dentro desse bloco, ele define como os elementos devem se comportar nessa situação. Isso evita que o usuário precise dar zoom ou role muito para os lados em telas pequenas, deixando a navegação muito mais confortável.
