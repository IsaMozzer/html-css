BREACKPOINTS MAIS USADOS

Os breakpoints são os tamanhos de tela que os desenvolvedores usam como referência para decidir quando mudar o layout do site com media queries. Eles são chamados assim porque representam o “ponto de corte” entre um tipo de dispositivo e outro. Os breakpoints mais comuns são: Até 600px: normalmente usados para celulares. É onde se faz a maioria das adaptações para telas pequenas, como reduzir fontes, empilhar blocos ou transformar o menu em ícone. Entre 601px e 1024px: faixa usada para tablets. O layout precisa ser mais espaçado que no celular, mas ainda não tão largo quanto no computador. Acima de 1024px: telas de computadores (desktops e notebooks). Aqui o site geralmente mostra o layout completo, com todos os elementos visíveis. Acima de 1440px: telas maiores, como monitores grandes ou TVs. Pode ser usado para aumentar margens, fontes e espaçamentos, se necessário.
